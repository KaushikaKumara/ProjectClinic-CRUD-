Pharmacy / Prescription

Common Medicines

Panadol

Description: Panadol is a common pain reliever and fever reducer.

Gaviscone

Description: Gaviscone is an antacid used to relieve heartburn and indigestion.

Aspirin

Description: Aspirin is a medication used to reduce pain, fever, or inflammation.

Benadryl

Description: Benadryl is an antihistamine used to relieve symptoms of allergy, hay fever, and the common cold.

Ibuprofen

Description: Ibuprofen is a nonsteroidal anti-inflammatory drug (NSAID) used to relieve pain, fever, and inflammation.

Loratadine

Description: Loratadine is an antihistamine used to treat allergies, hay fever, and hives.

Ranitidine

Description: Ranitidine is an H2 blocker that decreases the amount of acid created by the stomach.

Paracetamol

Description: Paracetamol is a medication used to treat pain and fever.

Tums

Description: Tums is an antacid used to relieve heartburn, acid indigestion, and upset stomach.

Cetirizine

Description: Cetirizine is an antihistamine used to relieve allergy symptoms such as watery eyes, runny nose, itching eyes/nose, and sneezing.
